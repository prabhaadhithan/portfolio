# GlowCare Skin Boutique

A static skincare ecommerce landing page built with HTML, CSS, and JavaScript. This project demonstrates a clean product storefront with shopping cart interactions, customer sections, and responsive design.

## Project Overview

This site is designed for a skincare brand and includes:

- Featured product catalog with product cards
- Add-to-cart functionality and cart modal
- Responsive layout for desktop and mobile
- Benefits, testimonials, and newsletter sections
- Simple visual design with clean branding

## Files

- `index.html` — Main homepage markup
- `styles.css` — Styling for layout, cards, buttons, and modal
- `scripts.js` — Cart behavior, cart updates, and newsletter form handling

## Setup

To run the site locally:

1. Open the `portfolio` folder.
2. Open `index.html` in your browser.

No additional build tools or dependencies are required.

## Usage

- Click **Add to cart** on any product to add it to the cart.
- Click the **Cart** button to view items and total cost.
- Remove items directly from the cart modal.
- Use the newsletter form to simulate an email subscription.

## Customization

To modify the site:

- Edit product details in `index.html`.
- Update colors, spacing, and typography in `styles.css`.
- Change cart behavior or add persistence in `scripts.js`.

## Notes

This is a static demo site intended for portfolio or landing page use. For a full ecommerce implementation, integrate a backend or payment provider.
